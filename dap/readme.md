## 代码来源

基于梁山派 [daplink]() 改了串口的代码，修复了原版串口数据偶尔异常的问题，把原版一直闪烁的led灯改为了串口指示灯。其中串口的代码来源于 abgelehnt 的[Tiny-DAPLink](https://github.com/abgelehnt/Tiny-DAPLink)

## 功能说明

- 支持SWD接口
- 支持虚拟串口
- 仅支持winusb通讯
